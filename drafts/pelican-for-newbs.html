Pelican for Newbs

Published: Wed 12 August 2020
By katie

In misc.

An earlier version of this site was made using jekyll, a very popular static site generator written in ruby. However, I'd recently been getting a lot of notifications from github about security issues associated with the whole set up. Rather dig into jekyll, I figured I'd check pelican out. I call this post "pelican for newbs" because it's dead easy. The bad news is though that most of the themes seem to be broken as of a recent update. For a list of the themes that are working, check out this github issue. Unfortunately but not surprisingly, the Pelican documentation isn't quite 100% clear - I'd guess because base python isn't exactly the first place you would turn to if you were in need of some front end. However, for us pythonistas, it's all we got so we gotta make the most of it!

This walk through assumes you know just a bit of bash (to work with files and folders and push to git).
